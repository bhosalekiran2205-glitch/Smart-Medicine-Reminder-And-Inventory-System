# AWS Lambda

This folder contains the AWS Lambda component used in the **Smart Medicine Reminder and Inventory System**.

## Purpose

The Lambda function is responsible for:

- Processing scheduled medicine reminders.
- Executing tasks triggered by Amazon EventBridge.
- Sending reminder emails through Amazon SES.
- Supporting the application's reminder workflow.

## AWS Services Used

- AWS Lambda
- Amazon EventBridge
- Amazon SES
- Amazon CloudWatch
