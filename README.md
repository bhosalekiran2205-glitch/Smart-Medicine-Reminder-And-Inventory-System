## About
Yeh ek web-based application hai jo medicine reminders aur inventory manage karne mein help karta hai.

## Features
- Medicine add karna
- Medicine reminders set karna
- Inventory track karna
- History dekhna
- Reports generate karna
- User login/register

## Technologies Used
- Python (Flask)
- HTML, CSS
- MySQL / SQLite

## How to Run
1. Repository clone karo:
   git clone https://github.com/bhosalekiran2205-glitch/Smart-Medicine-Reminder-And-Inventory-System.git

2. Dependencies install karo:
   pip install -r requirements.txt

3. App run karo:
   python app.py

4. Browser mein kholo:
   http://localhost:5000
